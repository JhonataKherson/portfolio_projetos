Projetos de Ciência de Dados

Construção de um programa de fidelidade com clusterização de clientes

Para esse projeto eu utilizei a lingugem Python, estatística e técnicas não-supervisionadas de machine learning para segmentar um grupo de clientes com base em suas características de performance de compra, a fim de selecionar grupos de clientes para formar um programa de fidelidade com objetivo de aumentar a receita da empresa. O resultado dessa solução , caso fosse implementada, seria de R$15MM de dólares de receita anual.

Ferramentas utilizadas:

  • Git, Gitlab e Github;
  • Python, Pandas, Matplotlib e Seaborn;
  • Jupyter Notebook;
  • K-Means, Hierarquical Clustering, DBScan;
  • AWS Cloud (EC2, S3, Postgres, SQLite);
  • Metabase Visualization.
  • SAIBA MAIS

Identificação de imóveis para compra e revenda a fim de maximizar o lucro
